It is possible to control plumber server without having to use `grpcurl` or another gRPC tool to talk to its gRPC API.

You can also "talk" to plumber by using the built-in `manage` command in the `plumber` CLI tool itself.

The `manage` sub-command allows you manage connections, relays and tunnels.

## Examples

```bash
$ plumber manage get connection

$ plumber manage get relay

$ plumber manage get tunnel
```

### Create a connection

```bash
$ plumber manage create connection kafka --name="My Kafka" --address "localhost:9092"
```
